# Introduction
Welcome to the documentation of FairML, a tool to build Machine Learning models and RL agents with certain desirable behavior.  
You can read more about this approach [here](https://aisafety.cs.umass.edu/paper.html).  

Also, read more about a comprehensive quickstart guide at [https://aisafety.cs.umass.edu](https://aisafety.cs.umass.edu).
